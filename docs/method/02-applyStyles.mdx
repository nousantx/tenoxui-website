# applyStyles

This is a method that handle the output of `applyStyle` function.

## Usage

```js
// selector
const selector = document.querySelector(".my-box");

// instance
const styler = new makeTenoxUI(selector);

// apply styles
styler.applyStyles("p-2rem")
```

Ok, just like that.

:::info
`applyStyles` only apply one class!
:::
