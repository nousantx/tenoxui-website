# applyMultiStyles

This is just upgrade from `applyStyles` :D

## Usage

```js
// selector
const selector = document.querySelector(".my-box");

// instance
const styler = new makeTenoxUI(selector);

// apply styles
styler.applyMultiStyles("box-100px bg-red br-0.125rem")
```

Ok, just like that.

:::info
`applyStyles` only apply one class!
:::
