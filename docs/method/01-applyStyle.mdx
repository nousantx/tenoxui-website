# applyStyle

`applyStyle` is the first method you will learn, this method used to apply a single style to element.

## Example

Let's add some element to html file :

```html
<div class="mybox box-50px"></div>
```

Add style :

```js
// selector
const selector = document.querySelector(".mybox");

// call tenoxui instance
const styler = new makeTenoxUI(selector);

// applying style
styler.applyStyle(
  "bg-image", // type will given value
  "https://tenoxui.web.app/img/tenoxui.svg", // background-image value
);
```
:::warning
This method is not recommended for applying background-image to an element, use `css variable` instead!

```css title="index.css"
:root {
  --myBack: /path/to/image;
}
```
```html title="index.html"
<div class="box-50px bg-image-[myBack]"></div>
```
:::
