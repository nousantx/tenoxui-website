# makeTenoxUI Class

`makeTenoxUI` class is responsible for applying styles to HTML element using defined class names.

```js title="tenoxui.js"
class makeTenoxUI {
  // TenoxUI constructor
  constructor(element) {
    this.element = element;
    this.styles = property;
  }
}
```

`makeTenoxUI` take a HTML element as argument. To use this class you can make this instance like this :

```js
new makeTenoxUI(element);
```

Let's make a simple example for using this instance :

```js title="index.js"
// defining element you want to use
const element = document.querySelector(selector);

// make instance of tenoxui
const styler = new makeTenoxUI(element);

// applying method
styler.methodExample();
```
