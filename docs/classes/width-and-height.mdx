# Width and Height üìê

How to define width and height of element?

:::info[Rules]

## Type Rules

<Tabs>
  <TabItem value="width" label="Width">

### Width Property

**`w-`** : Set `width` to the element.

**`w-mn-`** : Set `min-width` to the element.

**`w-mx-`** : Set `max-width` to the element.

</TabItem>

<TabItem value="height" label="Height">

### Height Property

**`h-`** : Set `height` to the element.

**`h-mn-`** : Set `min-height` to the element.

**`h-mx-`** : Set `max-height` to the element.

</TabItem>

</Tabs>

:::

:::tip

You can also use `% (percent)` as the unit. Such as `w-100%` and so on.`

:::
