## Types

### Width

- `w`: `width` property
- `w-mn`: `min-width` property
- `w-mx`: `max-width` property

### Height

- `h`: `height` property
- `h-mn`: `min-height` property
- `h-mx`: `max-height` property

### Box

There's a type called `box`, and it will set for both `width` and `height` at the same time.

## Simple Usage

```html
<div class="w-100px h-200px">...</div>
<div class="box-100px">...</div>
```
