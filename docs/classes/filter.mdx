## Type

Give filter effect using `filter` type.

## Usage

In tenoxui `v0.10+`, you can use the filter class like this :

```html
<div class="filter-[blur(3.7px)] ..."></div>
```

It's same as `backdrop-filter` and `transform` property.

In tenoxui `v0.9` and @tenoxui/property `v1.1`, you can use tge filter name directly inside your element like this :

```html
<div class="blur-10px grayscale-50%"></div>
```
