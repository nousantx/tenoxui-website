Let's learn about text rules, and make it easier to setting style into text element. You will learn all text and font type that available in TenoxUI.

## Text and Font Type

It's a few of class that most used when you write css:

- `fs` : Set `font-size` to the element.

- `fw` : Set `font-weight` to the element.

- `lh` : Set `line-height` to the element.

- `ls` : Set `letter-spacing` to the element.

- `ta` : Set `text-align` to the element.

- `tc` : Set `text-color` to the element.

- `td` : Set `text-decoration` to the element.

- `ti` : Set `text-indent` to the element.

- `tn` : Set `text-transform` to the element.

- `ws` : Set `word-spacing` to the element.

- `text-style` : Set `text-style` to the element.

- `white-space` : Set `white-space` to the element.

:::tip
If TenoxUI not provided the type you want, you can define it yourself using [addtype and defineProps](/docs/function/addtype-and-defineprops).
:::

## Some Example

```html title='index.html'
<header class="ta-center">
  <h1 class="fs-2rem fw-800">Hello World!</h1>
  <p class="fs-18px fw-600 lh-1.4 ls-1px">
    It's a nice day to write a few code.
  </p>
</header>
```
