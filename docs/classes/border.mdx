# Border Property ðŸš§

Add border to some element.


## Basic Type

This is some type you can use:

- `bdr` : `border`
- `br` : `border-radius`
- `bw` : `border-width`
- `bs` : `border-style`
- `bc` : `border-color`
- `border`: `border-color`. On `moreColor` only.
- `bw-top` : `border-top-width`
- `bw-bottom` : `border-bottom-width`
- `bw-left` : `border-left-width`
- `bw-right` : `border-right-width`
- `radius-tl` : `border-top-left-radius`
- `radius-bl` : `border-bottom-left-radius`
- `radius-tr` : `border-top-right-radius`
- `radius-br` : `border-bottom-right-radius`
- `radius-top`: [ `br-tl`, `br-tr` ],
- `radius-bottom`: [ `br-bl`, `br-br` ],
- `radius-left`: [ `br-tl`, `br-bl` ],
- `radius-right`: [ `br-tr`, `br-br` ],

## Basic Usage
```html title="html"
<div class="bw-8px br-5px bc-red bs-solid">
  ...
</div>
```
