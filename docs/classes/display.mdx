How to define and use display property using TenoxUI.

## Type

Changing `display` property value, you can use `d-{value}` type.

## Usage

```html title="html"
<div class="d-flex flex-parent-center">...</div>
```

:::warning[I can't use inline-block?]

You can, but wrap the value inside square bracket `[]` like this :

```html
<div class="d-[inline-block]">...</div>
```

:::
