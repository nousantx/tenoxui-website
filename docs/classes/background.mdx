Adding background through you element.

## Background Property

You can set `background` property by use `bg-` type, and it can recognize all background value. Such as, color, css variable, and also using image.

- `white-space` : Set `white-space` to the element.

:::tip
If TenoxUI not provided the type you want, you can define it yourself using [addtype and defineProps](/docs/function/addtype-and-defineprops).
:::

## Some Example

```html title='index.html'
<header class="ta-center">
  <h1 class="fs-2rem fw-800">Hello World!</h1>
  <p class="fs-18px fw-600 lh-1.4 ls-1px">
    It's a nice day to write a few code.
  </p>
</header>
```
