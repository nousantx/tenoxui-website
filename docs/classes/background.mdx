## Background Property

You can set `background` property by use `bg-` type, and it can recognize all background value. Such as, color, css variable, and also using image. Available properties :

- `bg` : `background`
- `bg-size` : `background-size`
- `bg-clip` : `background-clip`
- `bg-repeat` : `background-repeat`
- `bg-loc` : `background-position`
- `bg-loc-x` : `background-position-x`
- `bg-loc-y` : `background-position-y`
- `bg-blend` : `background-blend`
- More still maintained.
