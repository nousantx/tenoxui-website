# Overflow ⬆️

Despite its not ocassionally used, but gotchu.

## Types

- `over`: Set `overflow` property
- `over-x`: Set `overflowX` property
- `over-y`: Set `overflowY` property.

## Usage Example

```html
<div class="w-100px over-hidden">...</div>
```
