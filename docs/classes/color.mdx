In this page you will learn how to change color for text.

## Type

You can use `tc-` type to change the color of the text of the element.

## Usage Example

This simple code you can copy and try it :

```html title="Color example"
<h1 class="tc-red">Color Name</h1>
<h1 class="tc-#ccf654">Hex Color</h1>
<h1 class="tc-[rgb(245,122,217)]">RGB Color</h1>
<h1 class="[--color]-green tc-$color">CSS Variables</h1>
```
