# makeStyle function

`makeStyle` allows you to apply style by adding its className or new selector.

```js
makeStyle(selector, style)
```

## Usage

<p>Add some html code</p>

```html
<div class="new-class">
  <!-- ... -->
</div>
```

<p>Make a style using `makeStyle`</p>

```js
makeStyle(".new-class", "p-2rem bg-red");
```

:::tip

Don't forget to add your selector, you can use class, id, attribute, and others selector on first parameter.

:::
