# addType

This function will make new property and append it into curent `Property` and `Classes`.

## About

`addType` function will have 2 parameter, first parameter is the type that will called its property, and it always be `string`. Second parameter is the property you want to call using the type before, and it must be an `Array` and you can add two or more property that you can called using type you make.

```js
addType(type, property);
```

## Usage

```js
// single property
addType("p", "padding");
// more than one proeprties
addType("ph", ["paddingLeft", "paddingRight"]);
```
