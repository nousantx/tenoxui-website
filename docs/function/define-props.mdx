# `defineProps`

## About

`defineProps` function. Similiar to `addType` function, but it can make one or more property at once. The paramater needed must be an `Object`.

## Usage

```js
defineProps({
  string: string,
  // or
  string: [string, string],
});
```

Example :

```js
defineProps({
  m: "margin",
  fx: "flex",
  px: ["paddingLeft", "paddingRight"],
  "flex-parent": ["justifyContent", "alignItems"],
});
```
