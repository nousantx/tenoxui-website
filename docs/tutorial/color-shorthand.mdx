# Color Class

It's a bit hard to use color from `css variable`'s value. So, i recommend you to make a palette with light to dark color like this :

```css title="root.css"
:root {
  /* Primary: Gǎn Lǎn Huáng Olive */
  --primary-100: #f9fff2;
  --primary-200: #e4ffc1;
  --primary-300: #d4ff90;
  --primary-400: #cbff5f;
  --primary-500: #c7ff2e;
  --primary-600: #a2c90f;
  --primary-700: #7d9303;
  --primary-800: #545c00;
  --primary-900: #252600;

  /* Accent: Blue Blue */
  --accent-100: #f2f2ff;
  --accent-200: #bdc1fe;
  --accent-300: #8592f8;
  --accent-400: #4c67e8;
  --accent-500: #1842c9;
  --accent-600: #0835a1;
  --accent-700: #022b78;
  --accent-800: #00204f;
  --accent-900: #001226;
}
```

After that, you can make a function to generate color classNames and then use `makeStyles` function to generate the usable classNames :

```js
// store the output color
let color = {};

function Color(categories) {
  categories.forEach((category) => {
    for (let i = 100; i <= 900; i += 100) {
      // this is the className that you can use as the output
      const outputText = `.text-${category}-${i}`;
      // using `tc-` type to change the color value
      const textColorValue = `tc-[${category}-${i}]`;
      color[outputText] = textColorValue;
      // button background color generation
      const bgClassName = `.bg-${category}-${i}`;
      const bgColorValue = `back-c-[${category}-${i}]`;
      color[bgClassName] = bgColorValue;
    }
  });
}
```
